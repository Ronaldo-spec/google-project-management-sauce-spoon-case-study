# Report Security Note

This repository includes secured PDF versions of the Google Project Management case study report.

## Files

- `Project-Management_Ronaldo-Firmansyah_ID.pdf` - original Indonesian report in secured PDF format.
- `Project-Management_Ronaldo-Firmansyah_EN.md` - English portfolio report in Markdown format for GitHub review.
- `Project-Management_Ronaldo-Firmansyah_EN.pdf` - English secured PDF version for portfolio distribution.

## PDF Security

The secured PDF files were generated with owner-password protection and restricted permissions for copying, extraction, modification, and printing in compliant PDF viewers.

Important limitation: PDF permission restrictions cannot fully prevent screenshots, OCR, or conversion by tools that ignore PDF permissions. The protection is intended to reduce casual copying and modification while keeping the reports viewable for portfolio review.

## Repository Scope

Editable source documents such as DOCX files are intentionally not included. The repository is focused on review-ready reports and portfolio documentation.

## Source Context

The Indonesian report was prepared as part of Google Project Management coursework. The English report was adapted into a concise portfolio version for GitHub and LinkedIn review.
